MASTER UI/UX DESIGN PROMPT — GIGCREDIT MOBILE APPLICATION


OBJECTIVE


Design a clean, modern, fintech-grade mobile UI for the GigCredit application.


The UI must:


• Be consistent across all screens
• Be easy to understand within 20–30 seconds
• Clearly reflect verification + AI + trust system
• Adapt dynamically with backend processing states
• Provide smooth guided user experience


Design style:


• Minimal + modern fintech UI
• Soft gradients + clean cards
• Icon-driven + less text-heavy
• Smooth transitions and feedback


Primary theme:


Trust • Verification • Financial Growth • AI Intelligence
