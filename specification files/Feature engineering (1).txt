SECTION 2 � GLOBAL PREPROCESSING (compute before any pillar features)

BANK STATEMENT MONTHLY AGGREGATES:
  monthly_credit[m]         = sum(Credit Amount) for month m
  monthly_debit[m]          = sum(Debit Amount) for month m
  monthly_balance[m]        = last Running Balance of month m
  monthly_gig_credit[m]     = sum(Credit, tag=GIG_INCOME, month=m)
  monthly_emi_debit[m]      = sum(Debit, tag=EMI_DEBIT, month=m)
  monthly_utility_debit[m]  = sum(Debit, tag=UTILITY_DEBIT, month=m)
  months_covered            = count(distinct calendar months)
  REQUIRE: months_covered ? 6

INCOME SERIES:
  income_series             = [monthly_credit[1] � monthly_credit[n]]
  avg_monthly_income        = mean(income_series)
  std_monthly_income        = std(income_series)
  min_monthly_income        = min(income_series)
  max_monthly_income        = max(income_series)
  avg_closing_balance       = mean(monthly_balance[1..n])

UPI MONTHLY AGGREGATES (if statement available):
  upi_sent[m]               = sum(Amount, SENT, SUCCESS, month=m)
  upi_received[m]           = sum(Amount, RECEIVED, SUCCESS, month=m)
  upi_failed_count[m]       = count(transactions, FAILED, month=m)
  upi_total_count[m]        = count(transactions, SUCCESS, month=m)
  upi_utility_paid[m]       = sum(Amount, tag=UPI_UTILITY_PAYMENT, month=m)
  upi_gig_received[m]       = sum(Amount, tag=UPI_GIG_INCOME, month=m)

BOUNCE DETECTION (pre-label, used by F-P2-13):
  bounce_events             = count(bank rows where Description matches:
                              BOUNCE | BOUNCED | CHQ RETURN | CHQ RTN |
                              ECS RETURN | NACH RETURN | ACH RETURN |
                              INWARD RTN | OUTWARD RTN)

ATM DETECTION (pre-label, used by F-P2-15):
  atm_debit_count           = count(debit rows where Mode=ATM OR
                              Description matches: ATM|CASH WDL|WITHDRAWAL)
  total_debit_count         = count(all debit transaction rows)

SAVINGS CREDIT DETECTION (pre-label, used by F-P4-11):
  savings_debit_months      = count(months where any transaction tagged
                              SAVINGS_CREDIT with Debit > 0 exists)
  Tag SAVINGS_CREDIT when Description matches:
    SIP | RD | RECURRING DEPOSIT | PPF | MUTUAL FUND | ELSS

EMI DETECTION (pre-label, used by P2, P3):
  emi_rows                  = bank rows where Description matches:
                              EMI | LOAN | INSTALLMENT | NACH EMI | ECS EMI
  Tag: EMI_DEBIT

UTILITY MERGE (bank + UPI combined):
  For each utility bill month m:
    bank_utility_match[m]   = 1 if bank has matching debit else 0
    upi_utility_match[m]    = 1 if UPI has matching UPI_UTILITY_PAYMENT else 0
    utility_match[m]        = max(bank_match[m], upi_match[m])

FX CONVERSION:
  For international platform income (USD):
    amount_inr = amount_usd � RBI_reference_rate on payout date
    FX fallback = 83.0 (if RBI rate unavailable)
  Platform credits from PAYONEER / WISE / UPWORK PAYMENT tagged as
  UPI_FREELANCE_INCOME in UPI and GIG_INCOME in bank.
