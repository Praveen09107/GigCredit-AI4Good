# Step 6 - Government Schemes Inputs
Source: specification files/GIGCREDIT — USER INPUT COLLECTION SPECIFICATION.txt

## Inputs Asked from User

Scheme selection (all optional):
1. eShram Registration
2. PM-SYM (Pradhan Mantri Shram Yogi Maandhan)
3. PMJJBY (Pradhan Mantri Jeevan Jyoti Bima Yojana)
4. PMMY MUDRA Loan
5. Public Provident Fund (PPF)
6. Udyam/MSME participation (included as sixth scheme requirement)

Per-scheme inputs (where applicable):
7. eShram UAN Number + eShram card upload
8. PM-SYM pension account/subscriber number + registration proof upload
9. PMJJBY insurance certificate upload
10. PMMY loan account number + loan acknowledgement + no-due certificate upload
11. PPF account number + passbook first-page upload
12. Udyam/MSME registration number + MSME certificate upload

## Key User Actions

1. Select enrolled schemes from the scheme cards.
2. Enter scheme numbers/IDs where required.
3. Upload supporting proof document(s) per selected scheme.
4. Run OCR/verification for each selected scheme.
5. Submit after selected schemes are completed.

## Validation Notes

1. Entire step is optional; user may skip all schemes.
2. For every selected scheme, required fields and proof uploads must pass verification.
3. Bank cross-checks are expected for contribution/premium/repayment signals where scheme rules apply.
