# Step 4 - Utility Bills Inputs
Source: gigcredit_app/lib/ui/screens/steps/step4_utilities_screen.dart

## Inputs Asked from User

Mandatory utility modules:
1. Electricity bill uploads
2. LPG bill uploads
3. Mobile bill uploads

Optional utility modules:
4. Rent proof uploads
5. WiFi/Broadband proof uploads
6. OTT subscription proof uploads

## Key User Actions

1. Upload documents per utility module.
2. Click Verify per utility module.
3. Complete mandatory modules (Electricity, LPG, Mobile).
4. Submit using Continue to Step 5.

## Validation Notes

1. Mandatory modules require verification before step completion.
2. Optional modules can improve profile but are not mandatory.
