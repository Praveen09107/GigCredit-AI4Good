# Step 5 - Work Proof Inputs
Source: specification files/GIGCREDIT — USER INPUT COLLECTION SPECIFICATION.txt

## Inputs Asked from User (Dynamic by Work Type)

Platform Worker:
1. Vehicle Number
2. Platform proof upload (screenshots/documents)

Vendor/Seller:
1. SVANidhi ID (optional)
2. FSSAI Number (optional)
3. Vendor work proof upload

Skilled Tradesperson:
1. Skill Certificate ID (optional)
2. FSSAI Number (optional)
3. Trades work proof upload

Freelancer:
1. Freelance platform profile screenshot
2. Client invoice sample uploads (minimum one)
3. MSME Certificate upload (required as per latest requirement)
4. GST Registration Certificate (if applicable)

## Key User Actions

1. Fill applicable fields based on selected work type.
2. Upload relevant work proof document.
3. Either continue with verified data or choose Skip Step 5.
4. Submit using Continue to Step 6.

## Validation Notes

1. Validation logic changes per work type.
2. Freelancer flow must include MSME certificate for work identity compliance.
3. Freelancer invoice values are expected to be cross-checked against bank credits.
